# Requisitos

  - Node.js
  - NPM (Node Package Manager)


# Como executar o projeto

```bash
# clonar repositório
git clone https://github.com/marciosenaf/resposta-em-massa

instalar dependencies:
npm install

# executar o server
npm run server

# executar o client 
npm run client
```
